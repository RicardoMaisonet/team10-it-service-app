# Team 10 — IT Services Web App

A team capstone project building a cloud-hosted web application that offers IT services, with a Team Info page linking to each member's personal GitHub Pages resume site.

## Course
ITS 8 — Cloud Services Capstone

## Team Members & Roles
- **Eric** — Cloud Architect / Team Lead
  - Documentation, planning, architecture diagram, project tracking
- **Stephen** — Front-End Developer
  - Personal resume website, UI/UX, styling
- **Ricardo** — DevOps & Backend Engineer
  - Repo management, project structure, backend planning

## Project Goal
Build and deploy a multi-page web application showcasing IT services, with team member resume pages, hosted on Azure or AWS (TBD).

## Planned Tech Stack
- **Frontend:** HTML / CSS / JavaScript
- **Backend:** Azure Functions + Cosmos DB *(or AWS Lambda + DynamoDB)*
- **Hosting:** Azure Static Web Apps *(or AWS S3 + CloudFront)*
- **Source Control:** GitHub
- **DNS & HTTPS:** Azure DNS + CDN *(or AWS Route 53 + CloudFront)*

## Repository Structure
- `/frontend` — Static site files (HTML/CSS/JS)
- `/backend` — Serverless function code
- `/docs` — Architecture diagrams and documentation
- `/.github` — Workflows and CI/CD configs (future)
